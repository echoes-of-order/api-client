# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [3.1.2] - 2026-01-06

### Added
* Git workflow for semantic version tagging and npm publishing
* Coverage report script
* Cache dependency to release.yml
* Vitest config
* Dependency for improved functionality

### Changed
* Repository visibility changed from private to public and back to private
* Package.json version handling in git action updated
* Yarn install and yarn.lock management improved for workspace usage

### Fixed
* Issue with yarn.lock being committed
* Versions updated for consistency
* Test script issues resolved

### Removed
* Obsolete test script
* Obsolete comment for better code readability 

Note: No breaking changes, deprecations, or security fixes are included in this release.
